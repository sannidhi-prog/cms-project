# Project Writeup

For this project, I chose to deploy the Article CMS using Azure App Service instead of a Virtual Machine.

Azure App Service is a Platform as a Service (PaaS) offering that allows developers to deploy web applications without managing the underlying infrastructure. It automatically handles scaling, patching, and server management.

Using App Service made the deployment process faster and simpler compared to using a Virtual Machine. With a VM, I would need to configure the operating system, install dependencies, manage updates, and handle server maintenance.

Because this project focuses on deploying a web application quickly and efficiently, Azure App Service was the better choice.
